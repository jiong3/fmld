Profiling

    Create a global cargo profile called profiling, see below how.
    Compile with cargo build --profile profiling.
    Record with samply record ./target/profiling/yourrustprogram.

To create the profiling cargo profile, create a text file at ~/.cargo/config.toml with the following content:

[profile.profiling]
inherits = "release"
debug = true
